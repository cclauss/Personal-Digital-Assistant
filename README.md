# Claussoft Personal Digital Assistant

### Install on Linux
$ `sudo apt update`
$ `sudo apt install espeak ffmpeg libespeak1 portaudio19-dev`
$ `pip install -r requirements.txt`

### Install on macOS
$ `brew install portaudio`

### Run
$ `claussoft_pda/claussoft_pda.py`
